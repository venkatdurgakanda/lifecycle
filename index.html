Mood-Based Music Recommender – Flowchart

Start Application
User launches app or service
Capture User Input
Text, facial image, or voice input
Detect Mood
Use AI/ML to analyze emotion from input
Map Mood to Genre
Happy → Pop, Sad → Acoustic, Angry → Rock
Query Music Database
Fetch songs using Spotify or other APIs
Filter & Personalize
Apply user preferences and history
Recommend Songs
Display songs to user based on mood
Get Feedback (Optional)
User likes, skips, or rates music
Update Model / Profile
Improve suggestions based on feedback
End
Session complete or continue loop
